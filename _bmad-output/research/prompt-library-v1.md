# Book Club Prompt Library

A curated collection of discussion prompts designed for casual readers and social book clubs. These prompts prioritize connection and personal reflection over academic analysis.

---

## Category 1: Icebreakers (5 prompts)

*Low-pressure opening questions to get everyone comfortable sharing*

| # | Prompt | Best For |
|---|--------|----------|
| 1.1 | "On a scale of 1-5, how would you rate this book? No wrong answers!" | First meetup, any book |
| 1.2 | "What's one word that sums up your feeling about this book?" | Quick opener, shy groups |
| 1.3 | "Did you finish the book, or are you still reading?" | Honest check-in, removes pressure |
| 1.4 | "What made you want to read this book?" | Discovery, understanding motivations |
| 1.5 | "What's the best thing that's happened to you since our last meetup?" | Building community, warm-up |

---

## Category 2: Character Deep-Dive (5 prompts)

*Exploring the people in the story*

| # | Prompt | Best For |
|---|--------|----------|
| 2.1 | "Which character did you relate to the most, and why?" | Fiction, character-driven stories |
| 2.2 | "If you could have coffee with one character, who would it be?" | Light discussion, newer readers |
| 2.3 | "Did any character surprise you? In what way?" | Character development focus |
| 2.4 | "Who would you cast as the main character in a movie adaptation?" | Pop culture angle, fun prompt |
| 2.5 | "Which character would you least want to meet in real life?" | Sparking debate, lighter tone |

## Usage Notes

### For Organizers

- **Start with Icebreakers** — Always begin with a low-pressure question (1.1-1.5)
- **Pick 4-6 prompts per meetup** — 1 icebreaker + 2-3 main + 1 wrap-up works well
- **Let conversation flow** — Prompts are starting points, not tests
- **No wrong answers** — Remind members that personal interpretation is valid

### For App Features

- **Prompt suggestions by book genre** — Fiction, Nonfiction, Memoir, etc.
- **Difficulty indicators** — Light/Food-for-thought/Deep (for different group energies)
- **Share prompts in-app** — Let members see the agenda before meetup
- **Quick-select bundles** — Pre-curated prompt sets for common scenarios

| Scenario | Recommended Prompts |
|----------|---------------------|

| Heavy/emotional book | 1.2, 4.1, 4.2, 6.2 |
| Nonfiction/actionable | 3.4, 4.4, 5.2, 6.3 |
| Large group (8+) | 1.5, 2.4, 5.4, 6.1 (shorter prompts) |
| Small group (3-5) | Any combination works well |
