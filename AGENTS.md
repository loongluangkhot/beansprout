# AGENTS.md

This repository grants Codex standing permission to use subagents proactively.

## Codex delegation policy

- Codex may spawn subagents without asking for per-task approval when delegation can improve speed, parallelism, or quality.
- Codex should still keep the user informed of major delegation decisions and summarize subagent outputs before final delivery.
- If a task is sensitive, high-risk, or has non-obvious tradeoffs, Codex should pause and confirm with the user before proceeding.
